Algebra Formulas

Basic Algebraic Identities

(a + b)² = a² + 2ab + b²
(a - b)² = a² - 2ab + b²
a² - b² = (a + b)(a - b)

Cubic Identities

(a + b)³ = a³ + 3a²b + 3ab² + b³
(a - b)³ = a³ - 3a²b + 3ab² - b³
a³ + b³ = (a + b)(a² - ab + b²)
a³ - b³ = (a - b)(a² + ab + b²)

Quadratic Formula

ax² + bx + c = 0
x = [-b ± √(b² - 4ac)] / 2a

Arithmetic Progression (AP)

n-th Term: Tn = a + (n - 1)d
Sum of n Terms: S = n/2 [2a + (n - 1)d]

Geometric Progression (GP)

n-th Term: Tn = ar^(n - 1)
Sum of n Terms: S = a(1 - r^n) / (1 - r) for r ≠ 1
Sum to Infinity: S = a / (1 - r) for |r| < 1

Binomial Expansion

(a + b)ⁿ = Σ [nCk * a^(n-k) * b^k]
Where nCk = n! / [k!(n - k)!]
