Algebra

Quadratic Formula: \\( x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \\)
Example: Solve the quadratic equation \\( 2x^2 + 3x - 2 = 0 \\).
Here, \\( a = 2 \\), \\( b = 3 \\), and \\( c = -2 \\).
Using the quadratic formula: \\( x = \frac{-3 \pm \sqrt{3^2 - 4(2)(-2)}}{2(2)} \\).
Solution: \\( x = \frac{-3 \pm \sqrt{9 + 16}}{4} = \frac{-3 \pm \sqrt{25}}{4} = \frac{-3 \pm 5}{4} \\).
So, \\( x = \frac{-3 + 5}{4} = \frac{2}{4} = 0.5 \\) or \\( x = \frac{-3 - 5}{4} = \frac{-8}{4} = -2 \\).
Sum of Roots of a Quadratic Equation: \\( \alpha + \beta = -\frac{b}{a} \\)
Example: For the equation \\( 2x^2 + 3x - 2 = 0 \\), find the sum of the roots.
Using the formula \\( \alpha + \beta = -\frac{b}{a} \\), where \\( b = 3 \\) and \\( a = 2 \\):
Sum of roots: \\( \alpha + \beta = -\frac{3}{2} = -1.5 \\).

Geometry

Area of Circle: \\( A = \pi r^2 \\)
Example: Find the area of a circle with radius \\( r = 7 \\).
Using the formula: \\( A = \pi r^2 \\).
Solution: \\( A = \pi \times 7^2 = 49\pi \\approx 153.94 \\) square units.
Volume of Cylinder: \\( V = \pi r^2 h \\)
Example: Find the volume of a cylinder with radius \\( r = 3 \\) and height \\( h = 5 \\).
Using the formula: \\( V = \pi r^2 h \\).
Solution: \\( V = \pi \times 3^2 \times 5 = 45\pi \\approx 141.37 \\) cubic units.

Time and Work

Work Done: \\( W = \text{Rate} \times \text{Time} \\)
Example: If a worker completes 20 units of work in 5 hours, find the rate of work.
Using the formula: \\( \text{Rate} = \frac{\text{Work}}{\text{Time}} \\).
Solution: \\( \text{Rate} = \frac{20}{5} = 4 \\) units per hour.
Combined Work Rate: \\( \frac{1}{T} = \frac{1}{T_1} + \frac{1}{T_2} + ... \\)
Example: If two workers complete a task in 6 and 8 hours, find the time taken when they work together.
Using the formula: \\( \frac{1}{T} = \frac{1}{T_1} + \frac{1}{T_2} \\).
Solution: \\( \frac{1}{T} = \frac{1}{6} + \frac{1}{8} = \frac{4}{24} + \frac{3}{24} = \frac{7}{24} \\).
So, \\( T = \frac{24}{7} \\approx 3.43 \\) hours.

Speed and Distance

Speed: \\( \text{Speed} = \frac{\text{Distance}}{\text{Time}} \\)
Example: A car travels 120 km in 2 hours. Find its speed.
Using the formula: \\( \text{Speed} = \frac{120}{2} = 60 \\) km/h.
Relative Speed (When Two Objects are Moving in Opposite Directions): \\( S_{rel} = S_1 + S_2 \\)
Example: Two trains are moving towards each other with speeds 60 km/h and 40 km/h. Find their relative speed.
Using the formula: \\( S_{rel} = 60 + 40 = 100 \\) km/h.

Profit and Loss

Profit Percentage: \\( \text{Profit \%} = \frac{\text{Profit}}{\text{Cost Price}} \times 100 \\)
Example: A man buys an article for \\( 500 \\) and sells it for \\( 600 \\). Find the profit percentage.
Profit = \\( 600 - 500 = 100 \\).
Profit percentage = \\( \frac{100}{500} \times 100 = 20\% \\).
Loss Percentage: \\( \text{Loss \%} = \frac{\text{Loss}}{\text{Cost Price}} \times 100 \\)
Example: A man buys an article for \\( 800 \\) and sells it for \\( 600 \\). Find the loss percentage.
Loss = \\( 800 - 600 = 200 \\).
Loss percentage = \\( \frac{200}{800} \times 100 = 25\% \\).
